## 📥 How to Download & Install

Because Strive Private AI runs powerful server-grade intelligence natively on your desktop, installation is straightforward but requires double-clicking the correct file for your specific computer.

### Step 1: Download the App
Head to our **[Releases Page](https://github.com/Lee996677/strive-downloads/releases/latest)** and look under the "Assets" section.
- **If you are on a Mac:** Download the file ending in `.dmg` (e.g., `Strive.Private.AI_1.3.0_aarch64.dmg`)
- **If you are on Windows:** Download the Setup file ending in `.exe` (e.g., `Strive.Private.AI_1.3.0_x64-setup.exe`)

> **Note:** Please ignore any other raw files (like `.msi` or `.deb`). You only need the `.dmg` or the `-setup.exe`.

---

### Step 2: Bypassing OS Security (First Launch Only)
Because Strive Private AI is a cutting-edge independent application, your operating system will try to quarantine it by default since we do not currently force users through the corporate Apple App Store.

#### 🍏 For Mac Users ("App is damaged" error)
Apple's "Gatekeeper" security will display a scary message saying the app is damaged. It is not damaged; Apple is just blocking it. 
1. Open the downloaded `.dmg` and drag the **Strive Private AI** icon into your `Applications` folder.
2. Open your Mac's **Terminal** app (Press `Cmd + Space` and type "Terminal").
3. Paste exactly this command and hit Enter: 
   ```bash
   xattr -cr /Applications/"Strive Private AI.app"
   ```
4. You can now double-click the app in your Applications folder and it will permanently open flawlessly!

#### 🪟 For Windows Users ("Windows protected your PC")
Windows SmartScreen will display a blue warning screen because it doesn't recognize our publisher certificate yet.
1. Double-click the downloaded `.exe` installer.
2. When the blue warning screen appears, click **More info**.
3. Click **Run anyway**.
